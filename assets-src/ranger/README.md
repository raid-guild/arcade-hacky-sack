# Ranger Pose Source

`front-idle-source.svg` is copied from `brood-tapper/tapper-images/characters/ranger.svg`.

`generated-pose-sheet-v1.png` was generated with the built-in image generation tool from the local Ranger reference. It uses a flat magenta chroma-key background.

`generated-pose-sheet-v1-alpha.png` is the chroma-key removed version used by `assets-src/build-ranger-sprites.mjs`.

Prompt summary:

- Create a seven-pose pixel-art sprite sheet of the same Raid Guild Ranger.
- Poses, left to right: front idle, front kick, side idle, side spin, back idle, back kick, jump/header.
- Preserve the dark hood/armor, olive cloak, bow, pendant, face details, brown boots, and red soles.
- Use crisp pixel art, one horizontal row, shared baseline, no labels.
- Use a flat `#ff00ff` chroma-key background.

Manual art review is still required before launch.
